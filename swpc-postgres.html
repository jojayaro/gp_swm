SWPC to Postgres

Solar Wind Data Ingestion Pipeline

Project Overview

This Proof of Concept (POC) pipeline demonstrates the ingestion of solar wind data from the SWPC (Space Weather Prediction Center) API into a PostgreSQL database. It's designed to run periodically to keep the database updated with the latest solar wind measurements.

Key Features

  • Fetches solar wind data from the SWPC API
  • Processes and filters data based on timestamp
  • Stores data in a PostgreSQL database
  • Uses connection pooling for efficient database operations
  • Parallel processing of data using Rayon

Tech Stack

  • Language: Rust
  • Database: PostgreSQL
  • Libraries: tokio, reqwest, serde, rayon
