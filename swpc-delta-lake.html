SWPC to Delta Lake

High-Performance Solar Wind Data Ingestion

Project Overview

This project demonstrates a high-performance Data Engineering Proof of Concept (POC) for ingesting real-time solar wind data from the Space Weather Prediction Center (SWPC) into Delta Lake using Rust. It showcases efficient handling of non-standard API payloads and frequent data updates.

Key Features

  • Real-time data ingestion with 1-minute update intervals
  • Robust error handling with recovery mechanisms
  • Delta Lake integration for reliable and efficient data storage
  • High-performance processing using Rust
  • Parallel processing with Rayon

Tech Stack

  • Language: Rust
  • Data Format: Delta Lake
  • Libraries: deltalake, tokio, reqwest, serde, rayon, chrono
