AER ST1 Parser

High-Performance Data Processing for Alberta Energy Regulator

Project Overview

This high-performance Rust-based parser efficiently processes the Alberta Energy Regulator's ST1 data, which contains critical information on oil and gas exploration licenses. It showcases efficient handling of non-standard API payloads and frequent data updates.

Key Features

  • Asynchronous file downloading
  • Robust parsing algorithm for extracting structured data
  • CSV output generation for seamless integration with data processing pipelines
  • Comprehensive error handling
  • Memory-efficient processing of large datasets

Tech Stack

  • Language: Rust
  • Libraries: tokio, reqwest
  • Output Format: CSV
